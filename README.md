# Baekjoon & Algorithm Practice

백준(BOJ) 문제 풀이와 코딩테스트 대비 알고리즘 유형별 학습 저장소.

## 폴더 구성

### 알고리즘 유형별 학습
- `greedy/` — 그리디
- `implementation/` — 구현·시뮬레이션
- `dfs_bfs/` — DFS/BFS 탐색
- `sorting/` — 정렬
- `search/` — 탐색(이진 탐색 등)
- `shortestpath/` — 최단 경로
- `dinamic/` — 다이나믹 프로그래밍(DP)

### 백준 문제 풀이
- `baekjoon_solutions/` — 문제 번호별 풀이 노트북. `(미해결)`, `(공부중)` 접두어는 진행 상태 표시
- `python_solutions/` — 문제 번호별 `.py` 풀이
- `06 이후 알고리즘/` — 백준 단계별(입출력, 조건문, 반복문, 배열, 기하 등) 풀이
